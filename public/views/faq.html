General Questions

What does this tool actually do?

This tool is a comprehensive accessibility auditing platform. it uses a fleet of specialized AI agents to crawl your websites and identify WCAG compliance issues, specifically focusing on missing alt text, broken links, and semantic structure.

How do I start a new scan?

Navigate to the Dashboard and click "Start New Scan", or go to the Scopes tab and click the "Run Scan" button on a specific scope. You can also create new scopes by importing sitemaps.

How does the Coda integration work?

Once you provide a Coda API Token and Doc ID in the Settings, the system will automatically sync every completed audit report to your Coda document, creating new sub-pages for each scan.

Why do I only see one issue per element?

To keep reports clean and actionable, our Intelligent Post-Processor merges overlapping findings. If multiple agents identify issues on the same element, the system automatically prioritizes the most severe problem (e.g., a "Missing Alt" takes precedence over "Generic Alt").

Agent Documentation

Active Fleet Capabilities

Our agents are specialized in different WCAG criteria:

  • Alt Text Presence: Identifies images missing alt attributes or properly using empty alt text.
  • Alt Text Quality: Evaluates descriptive quality of alt text using contextual heuristics and accessibility best practices.
  • Color Contrast: Validates text readability against background colors to meet WCAG 4.5:1 thresholds.
  • Keyboard Accessibility: Ensures all interactive elements are reachable and operable without a mouse.
  • Focus Visibility & Order: Flags suppressed focus outlines and positive tabindex values.
  • ARIA Validation: Checks for correct usage of ARIA roles, states, and properties.
  • Touch Target Heuristics: Flags clickable elements that appear too small or icon-only.
  • Link Integrity: Validates broken internal and external links.
